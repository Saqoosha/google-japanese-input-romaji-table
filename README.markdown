Google日本語入力用ローマ字テーブル
====


内容
----

*  dvorak-jp.txt - DvorakJP配列用のローマ字テーブル


使い方
----

Google日本語入力の環境設定ダイアログを開き、ローマ字テーブルのカスタマイズボタンを押して、編集ボタンの「ファイルからインポート」を選び、`dvorak-jp.txt`を読み込みます。
